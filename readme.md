# Flathot
Like Superhot, but 2d.

## Features
    - Movement
    - Time moves when you move
    - Shooting
    - Enemy pathfinding
    - Collision detection
    - Scene loader
    - Enemy Shooting
    - Game loop (start menu, game, death screen)
    - Animations (still need to figure out how to switch between different animations)
